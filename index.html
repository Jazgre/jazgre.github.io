Desarrollar un Sitio Web

En esta página, aprenderás los conceptos básicos para desarrollar un sitio web utilizando HTML y CSS.

HTML

HTML (HyperText Markup Language) es el lenguaje estándar para crear páginas web. Utiliza etiquetas para estructurar el contenido en un documento.

Ejemplo básico de HTML:

!DOCTYPE html>
<html lang="es">
<head>
    <meta charset="UTF-8">
    <title>Título de la Página</title>
</head>
<body>
    <h1>Encabezado</h1>
    <p>Este es un párrafo.</p>
</body>
</html>

CSS

CSS (Cascading Style Sheets) es el lenguaje utilizado para estilizar un documento HTML. Define cómo los elementos HTML deben mostrarse en pantalla.

Ejemplo básico de CSS:

body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 0;
}

h1 {
    color: navy;
}

p {
    font-size: 16px;
}
